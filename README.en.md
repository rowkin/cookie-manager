# Cookie Manager with Daily Quote

Cookie Manager with Daily Quote is a powerful browser extension designed to help users effortlessly manage and manipulate website cookies. It not only provides an intuitive user interface but also integrates a daily motivational quote to inspire and motivate users. Whether you need to manage cookies or seek daily inspiration, this extension has got you covered.

## Installation Instructions

1. In your Chrome browser, navigate to `chrome://extensions/`.
2. Search for "Cookie Manager with Daily Quote" in the Chrome Web Store.
3. Click "Add to Chrome".
4. Wait for the installation to complete. Once installed, click the extension icon in the top-right corner to start using it.

## Usage Instructions

### Basic Operations

1. **View and Search Cookies**
   - Click "Get Cookies" to retrieve all cookies for the current website.
   - Use the top search bar to filter cookies in real-time.
   - View the total cookie count and detailed information.

2. **Cookie Management**
   - Add: Click the "Add Cookie" button.
   - Edit: Click the "Edit" button on each row.
   - Delete: Use the "Delete" button (for individual cookies) or "Delete All" (for all cookies).

3. **Pin Cookies**
   - Click the "Pin" button to pin important cookies.
   - Use "Pin All" to batch pin all cookies for the current page.
   - Pinned cookies will automatically apply to matching domains.

4. **Copy Cookies**
   - Simple format: name=value
   - Detailed format: complete JSON format (including all attributes).

### Advanced Operations

1. **Import/Export**
   - Export: Save current cookies as a JSON file.
   - Import: Batch import cookies from a JSON file.

2. **Auto-sync**
   - Automatically apply pinned cookies when switching tabs.
   - Maintain cookie state after page refresh.
   - Support retry mechanism for failed operations.

3. **Debugging Features**
   - Detailed operation status feedback.
   - Real-time notification of operation results.
   - Display of error messages.

## Precautions

1. Exercise caution when modifying cookies to avoid affecting website functionality.
2. It is recommended to confirm important data before exporting cookies.
3. The pin cookie feature will affect all pages under related domains.
4. Some websites may restrict cookie operations, rendering certain features unavailable.
