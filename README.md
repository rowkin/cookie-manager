# Cookie Manager Chrome Extension

Cookie Manager 是企平FE内部封装的 Chrome 浏览器扩展，用于管理和操作网站 Cookie。它提供了直观的用户界面和全面的 Cookie 管理功能。

## 功能特点

- 🔍 实时搜索 Cookie
- ➕ 添加新的 Cookie
- 🗑️ 删除单个或所有 Cookie
- 💾 导入/导出 Cookie 功能
- 🎨 优雅的渐变背景设计
- 💡 每日激励语录
- 📱 响应式设计，界面美观

## 使用说明

### 基本操作

1. **查看 Cookie**
   - 点击 "Get Cookies" 按钮获取当前网站的所有 Cookie
   - Cookie 信息将以表格形式显示，包含名称和值

2. **搜索 Cookie**
   - 使用顶部的搜索框实时过滤 Cookie
   - 输入关键词即可快速定位特定的 Cookie

3. **添加 Cookie**
   - 点击 "Add Cookie" 按钮添加新的 Cookie
   - 填写必要的 Cookie 信息（名称、值等）

4. **删除操作**
   - 单个删除：在 Cookie 列表中使用每行的删除按钮
   - 批量删除：使用 "Delete All Cookies" 按钮清除所有 Cookie

### 高级功能

1. **导入/导出**
   - 导出 Cookie：点击 "Export Cookies" 将当前 Cookie 导出为文件
   - 导入 Cookie：点击 "Import Cookies" 从文件中导入 Cookie

## 界面说明

- 主界面采用蓝色到红色的渐变设计
- 表格显示区域包含以下列：
  - Name：Cookie 名称
  - Value：Cookie 值
  - Actions：操作按钮（编辑/删除）
- 底部显示每日激励语录

## 注意事项

1. 修改 Cookie 时请谨慎操作，以免影响网站功能
2. 建议在导出 Cookie 前先确认重要数据
3. 插件需要相应的网站权限才能管理 Cookie
